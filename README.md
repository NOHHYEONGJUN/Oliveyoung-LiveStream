## 프로젝트 소개

- 3개 이상의 라이브 채널을 제공하는 **다중** 채널 라이브 서비스
- 클라우드 기반의 트래픽 처리를 통한 **안정적인** 라이브 서비스
- **AWS SaaS** 기반 ML 서비스 활용을 통한 개인 맞춤형 서비스
